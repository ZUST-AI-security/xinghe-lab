# 🚀 星河智安项目启动指南

## 📋 系统要求

**通用要求：**
- Python 3.8+ (推荐3.11)
- Node.js 16+ (推荐18)
- 现代浏览器 (Chrome、Firefox、Safari、Edge)

---

## 💻 Windows系统详细启动步骤

### 🔧 后端启动

#### 步骤1: 进入后端目录
```powershell
cd backend
```

#### 步骤2: 激活虚拟环境
```powershell
.\venv\Scripts\activate
```
> **注意**: 如果虚拟环境不存在，先创建：
> ```powershell
> python -m venv venv
> .\venv\Scripts\activate
> ```

#### 步骤3: 安装核心依赖
```powershell
pip install -r requirements.txt
```

#### 步骤4: 安装额外依赖 (解决兼容性问题)
```powershell
pip install email-validator python-multipart torch torchvision pillow
```

#### 步骤5: 安装可视化依赖 (跳过版本冲突)
```powershell
pip install matplotlib seaborn plotly scikit-learn tqdm opencv-python ultralytics
```

#### 步骤6: 启动后端服务
```powershell
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

> **预期输出**:
> ```
> INFO:     Started reloader process [xxxx]
> INFO:     ✅ 星河智安平台启动完成!
> INFO:     Application startup complete.
> ```

---

### 🎨 前端启动

#### 步骤1: 新开PowerShell窗口，进入前端目录
```powershell
cd web
```

#### 步骤2: 安装依赖 (使用legacy-peer-deps)
```powershell
npm install --legacy-peer-deps
```

#### 步骤3: 配置环境变量
```powershell
echo SKIP_PREFLIGHT_CHECK=true > .env
echo REACT_APP_API_BASE_URL=http://localhost:8000 >> .env
```

#### 步骤4: 启动前端服务
```powershell
npm start
```

> **预期输出**:
> ```
> Compiled successfully!
> You can now view xinghe-zhi-an-frontend in the browser.
> http://localhost:3000
> webpack compiled successfully
> ```

## ⚠️ 常见问题与解决方案

### 1. npm编译错误 - TypeScript冲突

**问题**: `ERESOLVE could not resolve` TypeScript版本冲突

**解决方案**:
```powershell
# 完全移除TypeScript相关依赖
npm uninstall typescript @types/node @types/react @types/react-dom --legacy-peer-deps

# 删除Vite配置文件 (如果存在)
rm vite.config.js

# 修复ESLint配置
# (见下方ESLint配置修复)
```

### 2. ESLint配置错误

**问题**: `Cannot resolve 'eslint/config'`

**解决方案**:
创建正确的 `eslint.config.js`:
```javascript
import js from '@eslint/js'
import globals from 'globals'
import reactHooks from 'eslint-plugin-react-hooks'
import reactRefresh from 'eslint-plugin-react-refresh'

export default [
  {
    files: ['**/*.{js,jsx}'],
    ignores: ['dist'],
    extends: [
      js.configs.recommended,
      reactHooks.configs.flat.recommended,
    ],
    languageOptions: {
      ecmaVersion: 2020,
      globals: globals.browser,
      parserOptions: {
        ecmaVersion: 'latest',
        ecmaFeatures: { jsx: true },
        sourceType: 'module',
      },
    },
    plugins: {
      'react-refresh': reactRefresh,
    },
    rules: {
      'no-unused-vars': ['error', { varsIgnorePattern: '^[A-Z_]' }],
      'react-refresh/only-export-components': [
        'warn',
        { allowConstantExport: true },
      ],
    },
  },
]
```

### 3. 依赖安全漏洞

**问题**: `30 vulnerabilities (9 low, 3 moderate, 18 high)`

**解决方案**:
```powershell
# 修复安全漏洞
npm audit fix --force

# 恢复react-scripts正确版本
npm install react-scripts@5.0.1 --legacy-peer-deps
```

### 4. 开发服务器配置错误

**问题**: `options.allowedHosts[0] should be a non-empty string`

**解决方案**:
```powershell
# 修复.env配置
echo "HOST=localhost" > .env
echo "SKIP_PREFLIGHT_CHECK=true" >> .env
echo "REACT_APP_API_BASE_URL=http://localhost:8000" >> .env

# 或在package.json中设置启动脚本
"start": "set HOST=localhost&& set DANGEROUSLY_DISABLE_HOST_CHECK=true&& react-scripts start"
```

### 5. numpy编译错误

**问题**: `ERROR: Failed to build 'numpy' when getting requirements to build wheel`

**解决方案**:
```powershell
# 跳过有问题的旧版本，使用已安装的新版本
pip install matplotlib seaborn plotly scikit-learn tqdm opencv-python ultralytics
```

### 6. pydantic-core编译错误

**问题**: `Cannot import 'maturin'` 或 Rust编译错误

**解决方案**:
```powershell
# 使用预编译版本
pip install pydantic pydantic-settings python-dotenv httpx
```

### 7. 端口占用问题

**问题**: `Something is already running on port 3000`

**解决方案**:
```powershell
# 终止所有node进程
taskkill /F /IM node.exe

# 重新启动前端
npm start
```

### 8. 依赖版本冲突

**问题**: 依赖包版本不兼容导致的编译失败

**解决方案**:
```powershell
# 使用legacy-peer-deps安装
npm install --legacy-peer-deps

# 或跳过严格版本检查
npm install package_name --upgrade
```

### 9. 构建工具冲突

**问题**: 项目同时存在Create React App和Vite配置

**解决方案**:
```powershell
# 选择Create React App (推荐)
rm vite.config.js

# 或迁移到Vite
npm install @vitejs/plugin-react vite --save-dev
# 然后修改package.json scripts部分
```

## 📝 开发注意事项

1. **虚拟环境**: 始终在激活的虚拟环境中工作
2. **端口管理**: 确保8000和3000端口未被占用
3. **依赖版本**: 遇到编译问题时优先使用预编译版本
4. **日志监控**: 启动后监控日志输出确认服务状态
5. **浏览器缓存**: 前端更新后可能需要清除浏览器缓存
